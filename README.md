# 📊 E-commerce Analytics Project

## 📌 Overview

This project focuses on analyzing e-commerce data to generate actionable insights and support data-driven decision-making. Using SQL for data analysis and Power BI/Tableau for visualization, the project uncovers trends in sales, customer behavior, and overall business performance.

---

## 🎯 Objectives

* Analyze revenue trends and sales performance
* Identify top-selling products and categories
* Measure customer retention and repeat purchases
* Evaluate seller performance and ratings
* Track key business KPIs for decision-making

---

## 🗂️ Dataset

The project uses a relational dataset with the following tables:

* Orders
* Order Items
* Customers
* Products
* Sellers
* Payments
* Reviews

---

## 🛠️ Tools & Technologies

* **SQL** – Data extraction and analysis
* **Power BI / Tableau** – Data visualization and dashboard creation
* **Excel (optional)** – Data preprocessing

---

## 📈 Key Insights

* Revenue trends over time
* Average Order Value (AOV) analysis
* Customer repeat purchase behavior
* Top-performing products and categories
* Seller performance and rating impact

---

## 📊 Dashboard Features

* Interactive filters and slicers
* KPI cards (Revenue, AOV, Repeat Rate, etc.)
* Trend analysis charts (line, bar, etc.)
* User-friendly and dynamic visualizations

---

## 🚀 Outcomes

* Improved understanding of customer behavior
* Identification of high-performing products and sellers
* Data-driven insights for business growth and strategy
